## Key Decisions
1. **Tailwind CDN + 原生 CSS** — 不用 CLI 构建，CDN 模式下 @apply 无效，所有样式写在 HTML class 或原生 CSS
2. **CoinGecko 风格** — 白底卡片，grid 表格布局，hover 效果，不用深色/glass
3. **Ollama 本地** — 免费、无延迟、无限调用，不用远程 API
4. **Cloudflare Tunnel** — 零成本公网部署，不需要租服务器
5. **异常阈值** — 5 倍均值（原 10 倍）、$5K 新地址（原 $50K），更适合 Mantle 链上交易量
6. **AI 阈值** — >$1000 触发 AI 分析
